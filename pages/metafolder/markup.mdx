# Handleiding schrijven in MarkUp

De docuemntatie gebruikt vrijwel dezeflde markup language als [Obsidian](https://guides.github.com/features/mastering-markdown/).

## Tekst

```
**bold**
*italic*
==highlight==
~~doorgestreept~~
# This is a heading 1 (titel)
## This is a heading 2 (ondertitel)
\ gebruik deze als je geen affect wilt..
```
